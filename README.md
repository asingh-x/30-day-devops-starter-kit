# 30-day-devops-starter-kit

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](./LICENSE)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](./CONTRIBUTING.md)

A free, open-source, hands-on DevOps curriculum designed to take a complete beginner to a **Level 1 DevOps Engineer** in 6 weeks.

No fluff. No endless theory. Just practical skills you can put on a resume.

---

## Who Is This For?

- Complete beginners with basic computer skills
- Developers who want to move into DevOps
- Anyone preparing for a junior/L1 DevOps role

## What You Will Learn

| Week | Topics | Goal |
|------|--------|------|
| [Week 1](./week-1/README.md) | Linux + Git | Work confidently in a terminal and manage code |
| [Week 2](./week-2/README.md) | Docker + Application Build | Package and run any application in a container |
| [Week 3](./week-3/README.md) | Kubernetes + Jenkins CI/CD | Deploy apps to a cluster and automate pipelines |
| [Week 4](./week-4/README.md) | AWS + Terraform + ECR | Provision and manage cloud infrastructure as code |
| [Week 5](./week-5/README.md) | Monitoring + Observability | Instrument apps with metrics, logs, and alerts |
| [Week 6](./week-6/README.md) | Security + Helm + GitOps | Production-ready deployments end to end |

## What You Need

- A computer with internet access
- Ubuntu/Linux or WSL2 on Windows (recommended)
- Free accounts: [GitHub](https://github.com), [AWS Free Tier](https://aws.amazon.com/free), [Docker Hub](https://hub.docker.com)
- 4–6 hours per day of focused practice

## How to Use This Repo

1. Start at [Week 1](./week-1/README.md) — do not skip ahead
2. Read the concept, then immediately do the exercise
3. Stuck? Check [Resources](./resources/) for cheatsheets
4. Complete the capstone project in Week 4 to build your portfolio

## Principles

- **Hands-on first** — every concept has a practice task
- **Understand why** — not just how
- **Build real things** — every week ends with a mini-project
- **Spaced repetition** — topics from earlier weeks appear in later exercises
